Linux is everywhere. From the world's fastest supercomputers and most web servers to the majority of smartphones, it underpins modern technology at scale. In the DevOps world specifically, most core tooling was designed with Linux first in mind: Docker started as Linux-only, Ansible controllers run on Linux, and Kubernetes control planes are Linux-based.

This course follows the KodeKloud Linux Basics curriculum and is structured so each chapter builds on the previous one — from navigating the shell through to managing storage and system services.

### What you'll learn

- Navigate the Linux CLI with confidence, using the shell as your primary interface
- Understand the core OS architecture: the kernel, boot process, runlevels, and file types
- Install and manage software using RPM, YUM, DPKG, and APT
- Master shell productivity: compression, searching, I/O redirection, pipes, and VI
- Understand networking fundamentals and troubleshoot connectivity issues
- Configure file permissions, ownership, ACLs, and SSH security
- Manage services with SYSTEMD — unit files, lifecycle commands, and custom services
- Work with storage: partitioning, filesystems, mounting, NFS, and LVM
